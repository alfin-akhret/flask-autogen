# flask-autogen

Simple Flask application generator and scafolding tools.

Installation
```sh
$ git clone https://github.com/alfin-akhret/flask-autogen.git
```


How to use
```sh
$ cd flask-autogen
$ python autogen.py install --target=your/project/dir
$ cd your/project/dir
$ pip install -r requirements
$ python run.py
```
